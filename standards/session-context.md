# Session context — compatibility reference

This file is retained for older scaffolds and is **not** the active Kiro session baseline.

The canonical, always-included Kiro baseline is [`.kiro/steering/crew-baseline.md`](../.kiro/steering/crew-baseline.md). Automatic role ownership and composition live in [`.kiro/steering/crew-roles.md`](../.kiro/steering/crew-roles.md), with native custom agents under [`.kiro/agents/`](../.kiro/agents/).

Do not inject this file through a SessionStart hook and do not copy routing tables into project `AGENTS.md` files. Install or update the active Kiro assets with `bin/init-kiro.ps1` or `bin/init-kiro.sh`.
